ICICI Policy No Format

Statement format: 4225/1000512039/00/0000 → segments: [insurer code] / [core] / [renewal] / [endo seq]
Master format: 100051203900 = core (1000512039) + renewal (00) concatenated
Endo matching: 4th segment 0001 → matches master Inst No E1 or M1 (the numeric part)
The ICICI transform mode handles all of this automatically.

Scoring & Boosts

Base scores come from the combo table in Match Rules → Default Rule → Score per Match Combination. On top of that: Exact PolNo match adds bonus points; Strong name match (≥85%) adds bonus; Exact premium match (0% difference) adds bonus; Year match (when optional) adds bonus. All configurable.

Premium Tolerance

Two tolerances run simultaneously — percentage (e.g. 2%) and absolute ₹ (e.g. ₹50). A match is counted if either tolerance is satisfied. This handles both large and small premiums fairly.

Share % Logic

If master Share % = 60 and master premium = ₹60, the full premium = ₹60 ÷ 0.60 = ₹100. The statement shows ₹100. With Share % applied, the match succeeds. Toggle off per insurer if not needed.
